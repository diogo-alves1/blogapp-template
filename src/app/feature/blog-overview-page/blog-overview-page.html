Willkommen zum Angular-Projekttemplate

Dieses Template ist dein Startpunkt für die Blog-Applikation im Modul IN353 Web Applications. Es enthält bereits das gesamte Tooling (ESLint, Prettier, Husky, Commitlint) und ein Angular Material Theme.

Quick Start

  1. ng serve — App lokal starten
  2. Ordnerstruktur erkunden: src/app/core, src/app/feature, src/app/shared
  3. ng generate component feature/my-feature — Neue Komponente erstellen
  4. npm run lint — Code prüfen
  5. npm test — Tests ausführen

Projektstruktur

src/app/
├── core/       ← Singleton Services, Guards, Interceptors
├── feature/    ← Lazy-loaded Feature-Komponenten
└── shared/     ← Wiederverwendbare Components, Pipes, Directives
